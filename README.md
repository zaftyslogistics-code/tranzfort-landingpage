# Tranzfort — India's Most Trusted Logistics Marketplace

**Website**: [www.tranzfort.com](https://www.tranzfort.com)

## About

Tranzfort is a logistics marketplace connecting verified truckers and suppliers across India. Book trucks, find return loads, and track shipments in real-time with zero commission and 100% transparency.

## Tech Stack

- **Vite** — Build tool
- **React 18** — UI framework
- **TypeScript** — Type safety
- **Tailwind CSS** — Utility-first styling
- **shadcn/ui** — Component library
- **Framer Motion** — Animations

## Getting Started

```sh
# Clone the repository
git clone https://github.com/zaftyslogistics-code/tranzfort-landingpage.git

# Navigate to the project
cd tranzfort-landingpage

# Install dependencies
npm install

# Start development server
npm run dev
```

## Build

```sh
npm run build
```
